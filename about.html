  • About The Server & The Game

To start playing on Moonlight Falls, you'll need to have Minetest Engine installed on your device.

Don't worry, Minetest is a free and open-source game, so you won't have to pay a penny to play it. If you haven't installed Minetest yet, you can download it from the official website by clicking here.

For smartphone or tablet users, you can get Minetest/Multicraft for Android and Multicraft for Apple devices. On Linux, you can install Minetest from your software manager or command line, while Windows users can download it directly from the official website.

Once you have Minetest installed, connecting to Moonlight Falls is simple and intuitive. From Minetest, simply search for "Moonlight Falls" in the server list or enter the server address (moonlight.edgy1.net) and port (30028) directly.

If you're using Multicraft, you can enter the invitation code "148.251.42.186:30028" to connect to our server. Remember to choose a username that you'd like to use when connecting to our server and create a secure password that you won't share with anyone.

So what are you waiting for? Start your adventure in our magical world today!
